## Error States

```
ERROR: Duplicate TIN
  Response: 409 Conflict
  Message: "Business with this TIN already exists"
  Action: User notified to use different TIN

ERROR: Missing Required Field
  Response: 400 Bad Request
  Message: "businessName, businessTin, ownerName... are required"
  Action: Form validation displays which fields are required

ERROR: Not Approved
  Response: (Redirected from form page)
  Message: "You must register and have an approved business..."
  Action: User redirected to register-business.html

ERROR: Owner User Not Found
  Response: 404 Not Found
  Message: "Owner user not found"
  Action: Server error - user should logout and login again
```
